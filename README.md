# Desafio-bindingForm

Aplicación de conocimientos del data binding en Vue Js a través de
las directivas para el llenado dinámico de un formulario para la creación de una tarjeta de crédito.

### Tecnologías utilizadas

- **HTML5.**
- **CSS3.**
- **Vue.Js.**
